# 🎬 FilmFitAI

FilmFitAI is an AI-powered movie recommendation web application that suggests movies based on user queries like genres, moods, or preferences.

It uses:
- ⚡ React (Frontend)
- 🚀 FastAPI (Backend)
- 🤖 Groq LLM + LangChain (AI Engine)

---

## 🌐 Live Demo

👉 https://film-fit-ai.vercel.app/  

---

## 📸 Preview

A simple Netflix-style UI where users can ask:
- "Suggest thriller movies"
- "Best comedy movies"
- "Top sci-fi films"

And receive AI-generated recommendations instantly.

---

## 🧠 Features

- 🎬 AI movie recommendation system
- 💬 Chat-based query interface
- ⚡ Fast response using Groq LLM
- 🌐 Fully deployed (Vercel + Render)
- 🎯 Simple and clean UI
- 🔥 Real-time API communication
